Our new game this week - MATCHSTRAVAGANZA

  1. How the game works!

  2. Press "Start" to begin the game and start the timer.
  3. Pick a card to reveal what is behind it.
  4. Pick a second card in the same manner.
  5. If they are the same image you made a successful match.
  6. If not they will turn back over and you must exercise your memory to find all the matches.
  7. Do this before the timer runs out to win the game!
